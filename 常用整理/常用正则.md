### 常用正则
-  邮箱校验
```
/^[A-Za-z0-9]+([._\-]*[A-Za-z0-9])*@([-A-Za-z0-9]+\.)+[A-Za-z0-9]+$/
```
- 手机号
```
/^1[34578]\d{9}$/
```
- 身份证
```
/^[1-9][0-9]{5}[1-2][0-9]{3}[0-1][0-9][0-3][0-9]{4}[0-9Xx]$/
```
- 由6-14位大小写字母、数字和标点符号组成(密码)
```
/^[a-zA-Z0-9_!@#$%^&*()_+]{6,14}$/
```
- 支持2-20位中英文、数字和下划线
```
/^[a-zA-Z0-9_\u4e00-\u9fa5]{2,20}$/
```
